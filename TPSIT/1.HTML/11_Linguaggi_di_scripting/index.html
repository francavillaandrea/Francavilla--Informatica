Benvenuto nel mondo della multimedialità

All’interno di una pagina HTML possono essere utilizzate anche istruzioni appartenenti ai cosiddetti linguaggi di scripting::
    1. vbScript
    2. javaScript
Il linguaggio vbScript era il default utilizzato da Internet Explorer
Il linguaggio javaScript è lo standard definito dal W3C (World Wide Web Consortium) oggi comunemente interpretato da tutti i browser.

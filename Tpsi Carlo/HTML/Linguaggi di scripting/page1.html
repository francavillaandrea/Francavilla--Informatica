Linguaggi di scripting

  1. Le istruzioni di scripting vengono scaricate dal browser client insieme alla pagina HTML richiesta
  2. Le istruzioni di scripting vengono pertanto eseguite sul PC client in seguito ad un’eventuale azione utente (ad esempio un click su un pulsante)
  3. Le procedure di scripting devono essere racchiuse tra i suguenti tag:

<Script Language = “javascript”>
<!—
…………………………………………………………
-->
</Script>
